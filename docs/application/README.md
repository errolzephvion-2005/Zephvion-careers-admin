# Application Management Module

## Overview
The Application module is designed to review, manage, and track candidate submissions for the various job postings at Zephvion.

## Routes
- `/dashboard/applications` - The main list view of all candidate applications.
- `/dashboard/applications/[id]` - Detailed view of a specific candidate's application.

## Features List
- **Application Tracking**: A comprehensive table showing all received applications across all jobs.
- **Filtering & Sorting**: Easily filter applications by job title, status, or date applied.
- **Candidate Details**: View full application data including resumes, cover letters, and contact information.
- **Status Management**: Update the status of an application (e.g., Pending, Reviewed, Interviewing, Rejected, Accepted).
- **Data Export**: Export candidate and application data to CSV or XLSX formats while preserving dashboard-applied filters.
- **Bulk Actions**: Perform operations like grouped deletion to manage candidate records cleanly.
